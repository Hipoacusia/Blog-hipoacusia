El hambito de la musica

Empece con FLStudio12,Garageband y MuseScore3. FL Studio es una estación de trabajo de audio digital con las características de editor de audio, secuenciador con soporte multipista y MIDI utilizado para la producción musical y desarrollado por la compañía belga Image-line Software. es uno de los DAW mas potentes y fluidos para mi, su interfaz es un poco complejo a entender en la primera ya que tienie muchas funciones. Tiene todo para hacer musica electronica,trap,lo-fi,dowmtempo,deephouse...etc.

GarageBand es una aplicación informática que permite al usuario crear piezas de música y podcast. Es desarrollado por Apple Computer para sus sistemas operativos iOS y macOS. GarageBand su interfaz es mucho mas sencillo y simple, tiene metodo de escritura musical o simplemente crear en un pianoroll. Cuenta con efectos muy buenos que se lo puedes otorgar a culquier instrumento. Cuenta con instrumentos un poco mas naturales, para hacer rock,pop,musica electronica, hip-hop, jazz...etc.

MuseScore es un programa de notación musical para Microsoft Windows, Mac OS X y Linux, que soporta una amplia variedad de formatos de archivo y métodos de entrada. Se publica como software libre y de código abierto bajo GNU General Public License. MuseScore es escritura musical pura, todo para hacer musica clasica, piano solista o orquestal. Tiene todo una libreria de sonidos de instrumentos naturalez y antiguos. Instrumentos de vientos de madera, Vientos de Metal, Cuerdas, Cuerdas frotados, Pianos, Organos y Coral. Tiene linias y dinamicas.
